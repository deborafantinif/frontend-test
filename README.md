## 💻 Projeto

Aplicativo web que recupera os dados disponíveis da [API do Studio Ghibli](https://ghibliapi.herokuapp.com/) e os exibi, permitindo a procura dos dados de forma interativa.

## 🤔 Execução

Você pode utilizar a aplicação por este [link](https://frontend-test-deborafantinif.vercel.app/films).

**Para inicialização do projeto dentro da máquina, siga os próximos passos**

- Clone o repositório:

  `git clone git@github.com:deborafantinif/frontend-test.git`

- Entre na pasta ghibli-search e instale as dependências:

  `npm install`

- Acesse a aplicação com o script start:

  `npm start`

- 💡Caso queira rodar a aplicação pelo docker, apenas clone o repositório e build o container:

  `docker-compose up --build`
  - A partir daí, a aplicação já está disponível em `http://localhost:3000/`.

## 🔧 Testes

- Para rodar os testes basta executar o scrip **test**:

  `npm run test`
